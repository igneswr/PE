Prerequisites and Information

  • NAT Loopback capable router recommended
  • If using CGNAT, hosting will NOT work
  • Router must support port forwarding
Example Port Forward: GAME SERVER Internal Port: 7777 External Port: 7777 SERVER ADDR Internal Port: 7778 External Port: 7778

Ports You Must Forward

Game Connection: 2838 Server Arbitrator: 2840-2843 Voice Chat: 2832 Sync Server: 2932

Server Configuration

  • Edit dedicated.cfg → server name, password, IP
  • Edit serverx.cfg → balancing settings

Testing Server

  • Launch server
  • Check server list
  • Connect

Non NAT Loopback Router Fix

  • Open system.cfg
  • Comment out:
net_customReduxClArbitratorAdr net_dedicatedServerArbitratorIP
