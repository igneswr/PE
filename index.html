How To Play

Zombie Classes

Catcher (Default) Balanced zombie class with reliable damage and mobility.
Mimic Can disguise as humans to deceive survivors.
Phantom Can turn invisible temporarily to ambush humans.
Bomber Explodes to deal massive damage to humans.

Human Perks

Rebellious (Default) Spawn with +1 armor.
Doubtful Carry 3 guns, third gun deals increased damage.
Lover Gain lifesteal when shooting zombies.
Safety Spawn with 5 armor and 1 heart. Armor and hearts regenerate.
Vanity Triple jump instead of double jump.
Leader Move faster and camouflage while crouching/sliding.
Hopeful Regenerate health and revive zombies into humans. 2 uses per round.

Host Your Own Server

Prerequisites and Information

  • NAT Loopback capable router
  • Non NAT Loopback routers work but joining method differs
  • CGNAT routers cannot host servers
  • Router capable of port forwarding
  • Some routers require manual internal/external port setup
GAME SERVER Internal Port: 7777 External Port: 7777 SERVER ADDR Internal Port: 7778 External Port: 7778

Server Port Forwarding

Game Connection: 2838 Server Arbitrator: 2840, 2841, 2842, 2843 Voice Chat: 2832 Sync Server: 2932

Server Configuration

  • Edit dedicated.cfg → name, password, IP
  • Edit serverx.cfg → balancing
